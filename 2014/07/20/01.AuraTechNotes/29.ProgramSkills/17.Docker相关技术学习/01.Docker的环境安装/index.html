Clone the repository by running Git in a container.

Docker映像是容器专用的文件系统。它提供了容器所需的所有文件和代码。

基于上一步中构建的图像启动容器。运行容器将启动具有私有资源的应用程序，与计算机的其他部分安全隔离。
