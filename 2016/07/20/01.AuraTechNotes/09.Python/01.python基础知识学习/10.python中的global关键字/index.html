在Python中，global关键字允许您在当前作用域之外修改变量。它用于创建全局变量并在局部上下文中对该变量进行更改。

全局关键字规则

Python中global关键字的基本规则是：

  • 当我们在函数内部创建变量时，默认情况下它是局部的。
  • 当我们在函数外部定义变量时，默认情况下它是全局的。您不必使用global关键字。
  • 我们使用global关键字在函数内部读写全局变量。
  • 在函数外使用global关键字无效
